# 模型选择指南

## 模型对比

### nano-banana-2（默认）

- **速度**: 快
- **画质**: 中等
- **消耗**: 低
- **适合**: 日常生成、快速迭代、批量出图
- **特点**: 综合表现均衡，英文提示词效果好
- **推荐场景**: 社交媒体配图、头像、快速草图、概念验证

### nano-banana-pro

- **速度**: 中等
- **画质**: 高
- **消耗**: 中高
- **适合**: 最终成品、高质量需求
- **特点**: 细节丰富，色彩还原好，适合复杂提示词
- **推荐场景**: 壁纸、海报、印刷品、展示用途、专业设计

### seedream-5.0-lite

- **速度**: 中等
- **画质**: 中高
- **消耗**: 中等
- **适合**: 创意设计、艺术风格
- **特点**: 色彩表现力强，艺术感突出
- **推荐场景**: 插画、创意设计、艺术作品、品牌视觉

### gpt-image-2

- **速度**: 中等
- **画质**: 高（摄影级）
- **消耗**: 高
- **适合**: 电商产品图、写实摄影、逼真场景
- **特点**: OpenAI 最新图像生成模型，摄影级写实输出，支持自定义尺寸（边长 16 的倍数，最长边 ≤3840，宽高比 1:3~3:1，总像素 655,360~8,294,400）
- **推荐场景**: 电商产品主图、商业摄影、写实人像、高端视觉内容
- **尺寸约束**: 预设 `1K`/`2K`/`4K`，或自定义 `WIDTHxHEIGHT` 格式

### seedream-4.5

- **速度**: 中等
- **画质**: 中高
- **消耗**: 中等
- **适合**: 通用场景
- **特点**: seedream 系列稳定版本
- **推荐场景**: 通用图片生成、设计素材

### qwen-image-plus

- **速度**: 中等
- **画质**: 中等
- **消耗**: 中等
- **适合**: 中文提示词、通用场景
- **特点**: 通义万相基础版，中文理解能力强
- **推荐场景**: 简单中文描述、快速出图

### qwen-image-2.0

- **速度**: 中等
- **画质**: 中高
- **消耗**: 中等
- **适合**: 中文提示词、中式美学
- **特点**: 通义万相升级版，中文语义理解更准确
- **推荐场景**: 中文描述生成、中国文化题材

### qwen-image-2.0-pro

- **速度**: 较慢
- **画质**: 高
- **消耗**: 高
- **适合**: 高质量中文提示词生成
- **特点**: 通义万相旗舰版，中文提示词理解最佳
- **推荐场景**: 复杂中文描述、中国文化题材、需要精确还原中文意境

## 选择决策树

```
用户需求
├── 快速出图 / 测试效果 → nano-banana-2
├── 电商产品图 / 摄影级写实 → gpt-image-2
├── 高质量成品
│   ├── 英文提示词 → nano-banana-pro
│   └── 中文提示词 → qwen-image-2.0-pro
├── 艺术/创意风格 → seedream-5.0-lite
├── 中国文化题材
│   ├── 快速 → qwen-image-2.0
│   └── 高质量 → qwen-image-2.0-pro
└── 未指定 / 不确定 → nano-banana-2（默认）
```
